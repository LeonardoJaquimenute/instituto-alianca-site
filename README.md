# Instituto Aliança – Site Institucional (Captação Social)

Aplicação web estática desenvolvida para uma Organização da Sociedade Civil (OSC), com foco em captação de recursos e apresentação institucional.

---

##  Descrição Técnica

Projeto construído utilizando arquitetura simples e escalável, com:

- Estrutura semântica em HTML5
- CSS organizado com abordagem Mobile First
- JavaScript Vanilla para interações dinâmicas
- Geração dinâmica de mensagens personalizadas para WhatsApp
- Scroll suave e navegação responsiva
- Estrutura modular de pastas

---

##  Principais Implementações

- Menu responsivo com toggle
- Scroll behavior suave
- Botões dinâmicos com data attributes
- Manipulação de DOM para geração de mensagens automáticas
- Estrutura pronta para futura integração com APIs de pagamento


---

##  Deploy

Projeto preparado para deploy em:

- GitHub Pages
- Hostinger

---

##  Melhorias Futuras

- Integração com gateway de pagamento (PIX / Stripe)
- Backend com Spring Boot
- Banco de dados para registro de doações
- Dashboard administrativo
- Integração com analytics
